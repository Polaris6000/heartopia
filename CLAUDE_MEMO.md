# 두근두근타운 홈페이지 — Claude 작업 메모

## 핵심 규칙 / 제약사항
- 코드 작성 시 주석 상세히 기입
- 반말, 간결한 어조로 소통

## 주요 파일 구조
```
/heartopia
├── index.html              # 진입점 (main.html로 리다이렉트)
├── 404.html                # 에러 페이지
├── admin.html              # 관리자 페이지
├── src/
│   ├── components/
│   │   ├── header.html     # 공통 네비게이션 헤더 (컴포넌트, <head> 없음)
│   │   └── footer.html     # 공통 푸터 (컴포넌트, <head> 없음)
│   ├── css/                # 스타일시트
│   ├── js/                 # 자바스크립트
│   ├── data/               # 데이터 파일
│   ├── locales/            # 다국어 파일
│   └── pages/
│       ├── main/main.html
│       ├── fish/           # 낚시 도감 + 물고기 찾기
│       ├── insect/         # 곤충 도감 + 벌레 찾기
│       ├── bird/           # 새 도감 + 새 찾기
│       ├── crops/          # 작물 도감
│       ├── recipe/         # 요리 레시피
│       └── contact/        # 문의
```
